# Android NFC 企业工具 App 页面原型说明文档（低保真评审稿）

## 1. 文档目的
用于产品、设计、开发联合评审，明确 V1 页面结构、交互流程、状态与跳转关系。

## 2. 信息架构

### 2.1 底部导航
- 首页
- 模板管理
- 日志审计
- 我的 / 设置

### 2.2 首页快捷入口
- 读卡
- 写卡
- 锁卡
- 解锁

### 2.3 统一原则
- 所有卡操作先进入通用扫描页
- 识别卡后按操作类型跳转对应页面
- 高风险操作必须二次确认并留痕

## 3. 页面原型说明

### 3.1 启动 / 环境检测页
**页面目标**：完成 NFC 能力、权限、网络、登录态等检查。

**页面结构**：
- 顶部：Logo / 应用名 / 版本号
- 中部：检测项列表（NFC、硬件、网络、登录态、策略同步）
- 底部：开始使用 / 重试检测 / 去设置

**跳转**：成功到首页；硬阻断失败到异常状态页。

### 3.2 首页
**页面目标**：展示主功能入口和运行概览。

**页面结构**：
- 顶部：企业名 / 当前用户 / 状态提示
- 中部：四宫格入口、今日概览、最近模板、最近日志
- 底部：Tab

**跳转**：四个操作入口均进入通用扫描页。

### 3.3 通用扫描页
**页面目标**：统一 NFC 扫描体验。

**页面结构**：
- 顶部：操作标题 + 返回
- 中部：扫描动效区
- 底部：状态 / 重试 / 取消

**分流**：
- 读卡 → 读卡结果页
- 写卡 → 写卡编辑页
- 锁卡 → 锁卡风险说明页
- 解锁 → 解锁验证页

### 3.4 读卡结果页
**展示内容**：UID、卡类型、数据内容、锁定状态、读取时间。

**按钮**：复制摘要、导出结果、继续读卡、返回首页。

### 3.5 写卡编辑页
**功能**：选择模板或手动输入，完成校验。

**页面结构**：
- 顶部：标题 + 目标卡摘要
- 中部：数据来源切换、表单区、校验提示区
- 底部：下一步确认

### 3.6 写卡确认页
**功能**：写入前最终确认。

**页面结构**：
- 顶部：风险级别标签
- 中部：目标卡信息、写入内容摘要、关键变更对比
- 底部：勾选确认、确认写卡按钮

### 3.7 写卡结果页
**展示内容**：结果状态、错误原因、写入块数、校验结果、审计流水号。

**按钮**：继续写下一张、返回首页、查看失败详情。

### 3.8 锁卡风险说明页
**功能**：锁卡前风险告知。

**页面结构**：
- 顶部：高风险标题
- 中部：风险说明、当前卡状态、锁定方式说明
- 底部：勾选确认 + 下一步

### 3.9 锁卡确认页
**功能**：执行前最后确认。

**页面结构**：
- 顶部：高风险标签
- 中部：卡摘要、操作员信息、影响范围
- 底部：确认词输入、确认锁卡按钮

### 3.10 锁卡结果页
**展示内容**：结果状态、锁卡时间、审计流水号、失败原因。

### 3.11 解锁验证页
**功能**：解锁前身份与权限验证。

**页面结构**：
- 顶部：标题
- 中部：卡摘要、验证方式、解锁理由输入
- 底部：提交验证

### 3.12 解锁结果页
**展示内容**：解锁结果、验证方式、操作员/审批人、审计号。

### 3.13 模板管理页
**功能**：模板列表、新建、编辑、复制、启停、删除。

**字段**：模板名称、版本号、适配卡型、更新时间、状态。

### 3.14 日志列表页
**功能**：按时间、操作类型、结果、操作员、卡 UID 查询日志。

### 3.15 日志详情页
**展示内容**：审计流水号、操作类型、操作员、设备信息、执行时间线、错误码。

### 3.16 异常状态页
**展示内容**：错误类别、错误码、建议操作、联系支持入口。

## 4. 全局组件规范
- 顶部导航：返回 / 标题 / 帮助
- 状态标签：成功绿、警告黄、失败红、处理中蓝
- 表单：必填 *、错误提示在字段下方
- 卡片组件：高级字段默认折叠
- 底部按钮区：高风险按钮用警示样式
- 脱敏规范：UID 默认掩码，密钥不明文展示

## 5. 高风险操作规范
适用页面：锁卡、解锁、删除模板、后续批量写卡。

规则：
1. 先风险说明，再最终确认
2. 明确后果，尤其是不可逆
3. 身份再验证
4. 防误触：确认词或长按确认
5. 执行中禁止重复提交
6. 全链路记录审计号
7. 失败必须给错误码与建议
8. 无权限提前拦截
